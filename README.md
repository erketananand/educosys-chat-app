# Vue Chat Application 
Vue 3 + Vite + Tailwind + Socket.io + Kafka + Redis


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Server start
```
npm run start
```
